Grand Theft Automation

An application designed to guide users through learning the principals of automation and basic javascript. Users advance through levels by programming a car to navigate progressively more difficult obstacles. As users encounter new automation problems they gain access to more vehicle features and unlock new javascript instructions they can give their car to avoid crashing.

Food Crawl

Given a destination, Food Crawl will find all the best restaurants along the way from where you are so you can eat without going too far astray. Uses the google maps API for directions and the Yelp API for restaurant information. All the information is collated into an intuitive UI built with AngularJS.
